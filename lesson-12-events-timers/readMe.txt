Создать проект CLOCK_DOM. Реализовать методами DOM показывающие текущее время и идущие в реальном времени часы по нижеуказанному макету.
Цифры часов не верстать «жёстко», создавать их и вычислять их позиции в цикле.
Никаких «волшебных констант» в коде не использовать — все константы вынести в начало скрипта с чётким документированием.

Как подступиться к созданию и позиционированию цифр:
 
0. Опишите константы - радиус циферблата, радиус кружка с цифрами. Опишите (создайте) пустой div-контейнер, внутри которого часы будут строиться программно.
 
1. Создайте внутри div-контейнера циферблат. Размер циферблата задаётся ранее описанными константами, остальные стилевые
свойства - CSS-классом. В частности, для циферблата должен быть установлен position:relative, чтобы мы могли позиционировать цифры и
стрелки именно относительно циферблата, а не относительно страницы.
 
2. Организуйте цикл от 1 до 12.
 
3. Посчитайте соответствующий очередной цифре угол в радианах, исходя из того что весь круг - это 2*Math.PI радиан.
 
4. Посчитайте координаты цифры - пример расчета лежит в папке с занятием. Исходите из того, что цифры позиционируются именно внутри div-а циферблата, т.е. относительно его левого верхнего угла.
 
5. Создайте div для цифры. Размер цифры задан константами в самом начале кода. Координаты цифры мы рассчитали на предыдущем шаге,
только учтите что мы рассчитали координаты центра цифры, а через style.left/style.top задаются координаты левого верхнего угла.
Остальные стилевые свойства цифры должны задаваться CSS-классом, в частности, для цифры должен быть установлен position:absolute.
В innerHTML цифры впишите собственно цифру.
 
6. Добавьте div цифры как дочерний к div циферблата.
Расположение цифр на часах - простой алгоритм:

var R=100; // радиус расположения цифр
var CX=500;
var CY=500; // координаты центра
for ( var H=1; H<=12; H++ ) // отображаемый час
{
  var A=H/12*Math.PI*2; // отображаемый угол в радианах
  var X=CX+Math.sin(A)*R; // проверяем - для угла=0 sin=0
  var Y=CY-Math.cos(A)*R; // проверяем - для угла=0 cos=1
  // итого цифра 12 (=0) окажется в X=CX, Y=CY-R.
}
